# 劳动合同解除的实务要点梳理

## 一、法律依据

用人单位单方解除劳动合同，主要依据《中华人民共和国劳动合同法》第三十九条（过失性解除）、第四十条（非过失性解除）及第四十一条（经济性裁员）。

## 二、过失性解除（第三十九条）

用人单位无需支付经济补偿，但需承担举证责任：

1. 试用期被证明不符合录用条件
2. 严重违反用人单位规章制度
3. 严重失职、营私舞弊，造成重大损害
4. 劳动者同时与其他单位建立劳动关系，对完成本单位工作造成严重影响
5. 以欺诈手段使对方在违背真实意思情况下订立合同
6. 被依法追究刑事责任

### 实务提示

- "严重违反"的认定标准需在规章制度中明确
- 解除前应充分调查取证，建议形成书面调查记录
- 应通知工会并听取意见

## 三、非过失性解除（第四十条）

用人单位需支付经济补偿（N或N+1）：

1. 劳动者患病或非因工负伤，医疗期满不能从事原工作
2. 劳动者不胜任工作，经培训或调岗后仍不胜任
3. 客观情况发生重大变化，合同无法继续履行

## 四、程序要件

| 步骤 | 要求 |
|------|------|
| 事实调查 | 收集证据，形成书面记录 |
| 工会通知 | 事先将理由通知工会 |
| 书面通知 | 出具《解除劳动合同通知书》 |
| 送达确认 | 确保劳动者收到解除通知 |
| 离职结算 | 结清工资，办理退工手续 |

## 五、法律后果

- 违法解除：支付赔偿金（2N）
- 合法解除（过失性）：无需支付经济补偿
- 合法解除（非过失性）：支付经济补偿（N或N+1）
