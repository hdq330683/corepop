使用说明

  • 文章在 WorkBuddy/本地写好 .md 文件后，拖拽上传即可
  • 标题留空时，系统会自动提取文中第一个 # 一级标题
  • 发布日期留空则使用当天日期
  • 排序序号：数字越小越靠前（如 1→排第1位），留空则自动排在板块末尾
  • 文件名会自动格式化为 YYYY-MM-DD-原标题.md
  • 点击发布后约 2-3 分钟自动上线，由 GitHub Actions 自动构建部署
  • 侧边栏目录完全自动更新，无需手动维护配置文件
